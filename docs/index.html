How it works

No manuals. No Stack Overflow. Just describe your problem.

1

Install TuxGenie

One curl command. All 29 built-in tools land on any Linux distro immediately.

2

Add your API key

Get a free key at console.anthropic.com. TuxGenie connects directly — your data never touches our servers. Pay-as-you-go via Anthropic (pennies per session).

3

Describe your problem

Type anything — "my wifi stopped working", "install chrome", or any shell command. Terminal commands run free — AI is only called when it truly helps.

Install on your distro

Pick your Linux distribution — get the exact install command for it.

💡
TuxGenie is free and open source. The app itself costs nothing to install or use.
AI features (diagnosis, fixes, script generation) use the Anthropic Claude API — you'll need a free API key from console.anthropic.com. Anthropic charges for AI usage (Haiku model ~$0.25 per million tokens — a typical session costs a fraction of a cent). Terminal commands bypass AI entirely and are always free.
